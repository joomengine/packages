### JCB! Custom Admin View
# Importer Engine (Queue) (importer)

## HTML:
```html
<div class="main-card">
	<?php echo Html::_('uitab.startTabSet', 'importerTab', ['active' => 'importer', 'recall' => true]); ?>
		<?php echo Html::_('uitab.addTab', 'importerTab', 'importer', Text::_('Importer', true)); ?>
			<div class="row">
				<div class="col-md-12">
					<?php echo $this->loadTemplate('importerarea'); // change this to the layout of your custom importer area ?>
				</div>
			</div>
		<?php echo Html::_('uitab.endTab'); ?>
		<?php echo Html::_('uitab.addTab', 'importerTab', 'history', Text::_('Queue', true)); ?>
			<div class="row">
				<div class="col-md-12">
					<?php echo $this->loadTemplate('importerqueue'); // change this to the layout of your custom importer queue ?>
				</div>
			</div>
		<?php echo Html::_('uitab.endTab'); ?>
	<?php echo Html::_('uitab.endTabSet'); ?>
</div>
```
